# 텍스트 파일 vs 이진 파일

## 텍스트 파일 (CSV 등)

- 사람이 직접 읽고 편집 가능
- 메모장, 엑셀 등 다양한 프로그램에서 열 수 있음
- 다른 언어/시스템과 호환 쉬움
- 모든 값이 문자열이라 읽을 때 타입 변환 필요
- 파일 크기가 상대적으로 큼

## 이진 파일 (pickle 등)

- 사람이 직접 읽기 어려움
- 파이썬 전용이라 다른 언어에서 읽기 어려움
- 파이썬 객체를 그대로 저장하므로 읽을 때 타입 변환 불필요
- 파일 크기가 상대적으로 작음
- 읽기/쓰기 속도가 빠름

## 결론

데이터를 외부와 공유하거나 사람이 직접 확인해야 할 때는 텍스트 파일,
파이썬 내부에서만 빠르게 저장/불러오기 할 때는 이진 파일이 적합합니다.

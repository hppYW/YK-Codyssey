# YK-Codyssey

학교 수업 과정인 코디세이(Codyssey) 학습 과정의 파이썬 개발 저장소입니다.
매 문제와 제약 사항은 별도로 안내됩니다.

## 개발 환경

- Python 3.x

## 기본 제약 사항

- Python 기본 제공 명령어만 사용 (별도 라이브러리/패키지 사용 금지)
- 경고 메시지 없이 모든 코드가 실행되어야 함

## 코딩 스타일 (PEP 8 준수)

- 문자열은 기본적으로 `' '`(작은따옴표) 사용
  - 문자열 내에 `'`가 포함되는 경우에만 `" "`(큰따옴표) 사용
- 대입문의 `=` 앞뒤에 공백 추가 (예: `foo = (0,)`)
- 들여쓰기는 공백(space) 사용
- 코드에 간단한 주석 추가
